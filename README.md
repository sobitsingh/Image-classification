# Image-classification
Fashion-MNIST is a dataset of Zalando's article imagesâ€”consisting of a training set of 60,000 examples and a test set of 10,000 examples. Each example is a 28x28 grayscale image a total of 784 pixels, associated with a label from 10 classes.Each pixel has a single pixel-value associated with it, indicating the lightness or darkness of that pixel, with higher numbers meaning darker. 
The first column consists of the class labels and represents the article of clothing. The rest of the columns contain the pixel-values of the associated image.
Labels are as follows :-
	0 T Shirt/top 		1 Trousers 		2 Pullover		3 Dress 	4 Coat		
	5 Sandal		6 Shirt			7 Sneaker		8 Bag		9 Ankle Boot
